# Lessons

`MataMentorPi` keeps the approachable TurboPi V2 teaching style while adding MentorPi-native perception, mapping, navigation, and AI topics.

The 20 lessons are organized as a full progression:

1. movement and safety
2. robot expression
3. sonar and infrared sensing
4. color vision and QR codes
5. tracking and line following
6. lidar obstacle avoidance
7. lidar follow and guard
8. depth camera basics
9. point clouds and spatial decisions
10. SLAM mapping
11. map saving and localization
12. navigation goals and waypoints
13. MediaPipe hands
14. pose and gesture control
15. YOLO object detection
16. autonomous driving
17. multi-robot coordination
18. voice and LLM missions
19. vision-language scene understanding
20. MentorPi capstone missions

Each lesson folder contains a `lesson.json` manifest and a student-facing markdown handout.
